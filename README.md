## Products & Platforms

### Qwerty Chat
**Platform:** Android  
**Technology:** Bluetooth Low Energy (BLE)  
**Architecture:** Decentralized  

Qwerty Chat is a public-use Android application that enables communication using Bluetooth LE technology. It is designed to function:

- **Without an internet connection**
- **With an internet connection**, where available, while still leveraging BLE capabilities  

Key characteristics:
- Decentralized communication model  
- Peer-to-peer message exchange  
- Operates in environments with limited or no network infrastructure  
- Suitable for proximity-based and ad-hoc communication scenarios  

Qwerty Chat is built for accessibility, resilience, and practical real-world use.

---

### TARK
**Category:** Corporate & Enterprise Platform  
**Model:** Decentralized SaaS  
**Focus:** High-security communications  

TARK is a corporate-grade decentralized system offering Software-as-a-Service (SaaS) solutions to organizations that require **military-grade communication security**.

Key features include:
- Decentralized system architecture  
- Enterprise-level access control  
- Secure communication channels  
- Designed for sensitive and mission-critical operations  

TARK is intended for organizations where confidentiality, integrity, and availability are non-negotiable.
